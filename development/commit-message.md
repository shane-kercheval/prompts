Please write a commit title and message based on the local changes. Use tools to get the list of changed files and the diffs.

- DO NOT perform the commit, only write the title and message.
- DO NOT include an author or "generated by" lines.
- Be concise but informative. Do not be overly verbose.
- Follow commit message best practices.
